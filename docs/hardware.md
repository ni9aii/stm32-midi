## Матрица клавиш

Матрица 5×5, 25 клавиш. Каждая клавиша имеет последовательный диод `1N4148`, поэтому можно безопасно сканировать строки по одной.

### Строки

Строки выбираются 74HC238. Выходы 74HC238 активны низким уровнем, поэтому выбранная строка имеет логический `0`.

| Строка | 74HC238 выход | Клавиши | MCU address |
|---:|---|---|---|
| 0 | Y0 | SB1–SB5 | PB0 = A0 |
| 1 | Y1 | SB6–SB10 | PB1 = A1 |
| 2 | Y2 | SB11–SB15 | PB11 = A2 |
| 3 | Y3 | SB16–SB20 | PB0/PB1/PB11 = `0b011` |
| 4 | Y4 | SB21–SB25 | PB0/PB1/PB11 = `0b100` |

### Столбцы

Столбцы коммутируются 74HC151. Выбранная линия столбца заводится на `PB13`. Столбцы подтянуты к +3V через R13–R17, поэтому нажатая клавиша читается как `0`.

| Столбец | 74HC151 вход | Клавиши | Pull-up | MCU select |
|---:|---|---|---|---|
| 0 | I0 | SB1, SB6, SB11, SB16, SB21 | R13 | PA1 = S0 |
| 1 | I1 | SB2, SB7, SB12, SB17, SB22 | R14 | PA2 = S1 |
| 2 | I2 | SB3, SB8, SB13, SB18, SB23 | R15 | PB12 = S2 |
| 3 | I3 | SB4, SB9, SB14, SB19, SB24 | R16 | PA1/PA2/PB12 = `0b011` |
| 4 | I4 | SB5, SB10, SB15, SB20, SB25 | R17 | PA1/PA2/PB12 = `0b100` |

Входы `I5–I7` на 74HC151 не используются.

## USB-MIDI

Прошивка использует USB Audio/MIDI class descriptor:

- AudioControl interface 0 с class-specific header.
- MIDIStreaming interface 1.
- Один Bulk IN endpoint `0x81`, max packet 64 байта.
- Один embedded MIDI IN jack и один embedded MIDI OUT jack.

Текущая прошивка только передает события от клавиатуры в USB-хост.

## Замечания для проверки на железе

1. Если часть клавиш читается зеркально, проверить порядок address/select lines у 74HC238/74HC151 на реальной плате.
2. Если все клавиши инвертированы, проверить ориентацию диодов и назначение `Y`/`W` у 74HC151.
3. Если USB не определяется, проверить питание +3.3V, 8 MHz HSE и наличие pull-up на D+.
